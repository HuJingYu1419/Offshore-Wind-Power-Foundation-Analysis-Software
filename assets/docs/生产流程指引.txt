(1) 用Vite构建生产版本：npm run build
(2) 用Tauri打包生产可执行文件：npm run tauri:build
(3) 可选测试环节：npm run tauri:dev
(4) 实时查看软解和检查报错：npm run dev 

静态资源准备：
在根目录创建一个public/，然后把assets/pics复制到public/中，这样生产的软件才有图片这些静态资源。


配置vite和tauri的方法（前提条件：已安装Node.js）
1、npm init -y   #初始化构建package.json（需要自己调整里面的具体内容）
2、在根目录创建和配置vite.config.js
3、npm install   #安装依赖
4、npx tauri init   #初始化tauri（需根据交互式指引完成配置）
5、微调：src-tauri/tauri.conf.json（其中注意需改identifier属性的值，若使用默认值会被阻止生产，因为发布版本要求设置为专用的标识）
6、其他测试：
# 1. 检查配置文件是否正常
npx tauri info

# 2. 启动开发环境
npm run tauri:dev

# 3. 构建生产版本(这行命令会自动构建dist/，其中public/的资源会自动迁移到dist/下)
npm run tauri:build
